1️⃣ What is the difference between var, let, and const?
--->>> ANS:
VAR-->> var was more commonly used before ES6. It is function-scoped, can be re-declared and reassigned, and it is hoisted.

LET-->> let was introduced in ES6. It is block-scoped, cannot be re-declared in the same scope, but can be reassigned. It is hoisted, but the variable remains in the Temporal Dead Zone (TDZ) until its declaration is executed.

CONST-->> const was introduced in ES6. It is block-scoped, cannot be re-declared, and cannot be reassigned.

2️⃣ What is the spread operator (...)?
--->>> ANS: The spread operator is a JavaScript syntax (...) that expands the elements of an array or the properties of an object. Those elements or properties can then be extracted and used elsewhere, such as in new arrays, objects, or function arguments.

3️⃣ What is the difference between map(), filter(), and forEach()?
--->>> ANS: map() operates on each element of an array and returns a new array with the transformed values. filter() returns a new array containing only the elements that meet a specified condition. forEach() simply iterates over each element and performs an action but does not return a new array. 

4️⃣ What is an arrow function?
--->>> ANS: Arrow functions are a shorter syntax for writing functions in JavaScript, introduced in ES6. They allow you to write functions with less code while keeping the same behavior in most cases.

5️⃣ What are template literals?
--->>> ANS: Template literals are a modern way of writing strings in JavaScript, introduced in ES6. They use backticks (``) and allow you to easily insert variables or expressions inside a string using the ${...} syntax.
